WISHBONE System-on-Chip (SoC) Interconnection Architecture for Portable IP Cores¶

Stewardship

This specification is maintained by Richard Herveille (rherveille@opencores.org) and hosted by the Free and Open Source Silicon Foundation. Questions, comments and suggestions about this document are welcome and should be directed to the mailing list (https://lists.librecores.org/listinfo/discussion) and the GitHub repository (https://github.com/fossi-foundation/wishbone).

These specifications are intended to guarantee compatibility between compliant IP-cores and to improve cooperation among different users and suppliers.
